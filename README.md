# 📊 Power BI Retail & Sales Analytics Project  
*A Data Technician Bootcamp Project*

This Power BI project showcases my ability to transform raw retail and sales data into clear, interactive, and insight‑driven reports. Developed during my **Data Technician Bootcamp**, the project highlights skills in data cleaning, modelling, DAX, and data storytelling through dynamic visualizations.

## 🚀 Project Overview

The project analyses retail and sales performance across products, stores, and time periods. Using Power BI, I built interactive reports that help users explore:

- Sales by product category  
- Monthly and seasonal trends  
- Store‑level performance  
- Customer purchasing behaviour  

The reports guide stakeholders from high‑level insights to deeper analysis through intuitive navigation and interactive controls.

## 🛠️ Key Skills Demonstrated

### 🔧 Data Transformation with Power Query
- Cleaned and shaped raw datasets  
- Removed duplicates, corrected data types, and standardised formats  
- Merged and appended tables to build a reliable data model  
- Created custom columns to support deeper analysis  

### 📐 DAX Calculated Columns & Measures
- Built calculated columns for segmentation and categorisation  
- Created DAX measures for KPIs including:  
  - Total Sales  
  - Year‑over‑Year Growth  
  - Profit Margin  
  - Category Share  
- Applied time‑intelligence functions for trend analysis  

### 🎛️ Interactive Reports (Slicers & Filters)
- Added slicers for date, category, region, and store  
- Used filters to enable drill‑down and cross‑highlighting  
- Designed clean, user‑friendly layouts for smooth exploration  

### 📊 Visualizations
A range of visuals were used to communicate insights effectively:

- **Bar charts** for category and store comparisons  
- **Line charts** for time‑series trends  
- **Pie charts** for distribution analysis  
- **Map visuals** for geographic performance  
- **KPI cards** for high‑level metrics  

## 📈 Data Storytelling Focus

The project emphasises **data storytelling**, helping users understand:

1. **What is happening** — overall sales performance  
2. **Why it’s happening** — category, region, and customer insights  
3. **What it means** — opportunities for improvement and growth  

Each report is structured to reveal insights progressively, making the data accessible and actionable.
